# Tablet — Экран переговорной комнаты

## Зона ответственности
Веб-приложение для планшета, установленного у двери переговорной. Показывает статус комнаты и расписание. Доступно на https://app.corpmeet.uz

## Стек
- React 18, Vite

## Структура
```
tablet/
├── index.html
├── src/
│   ├── main.jsx       # Точка входа
│   └── App.jsx        # Корневой компонент
```

## Правила
- Интерфейс должен быть крупным и читаемым издалека
- Основной сценарий: видно свободна комната или занята (цветом)
- Автообновление данных (polling или WebSocket к /api/)
- Минимум интерактива — планшет висит на стене, не для активного ввода
- Те же правила что для frontend: функциональные компоненты, хуки, fetch

## Запуск локально
```bash
cd tablet
npm install
npm run dev
```

## UX-требования
- Зелёный фон = свободна
- Красный фон = занята
- Показывать текущее и следующее бронирование
- Полноэкранный режим без скроллов
